dni-validator-peru

Validación offline de DNI y RUC peruanos con módulo-11 SUNAT. Zero dependencies. Mantenido por Securex.

Ver en GitHub npm securex.pe

Por qué existe

Si tu app pide DNI o RUC en algún punto del flujo (KYC, onboarding, facturación, ecommerce con factura electrónica), tarde o temprano te toca validar y consultar SUNAT, RENIEC o un proveedor que envuelva ambos.

Antes de gastar una llamada API por cada documento que entra, vale la pena validar offline: muchos errores son tan triviales como un dígito de menos, un espacio metido al inicio, o un usuario tipeando 12345678 a ver si pasa.

Validar localmente te ahorra latencia, costo (si pagas por consulta) y rate limits.

Instalación

npm install dni-validator-peru

Uso

import {
  isValidDNI,
  isValidRUC,
  validateDocumento,
  tipoContribuyente,
} from 'dni-validator-peru';

isValidDNI('72345678');         // true
isValidDNI('00000000');         // false (secuencia trivial)

isValidRUC('20602431216');      // true (Grupo Securex S.A.C.)
tipoContribuyente('20602431216'); // 'juridica'

validateDocumento('72345678');    // { tipo: 'DNI', valid: true }
validateDocumento('20602431216'); // { tipo: 'RUC', valid: true, subtipo: 'juridica' }
validateDocumento('abc');         // { tipo: null, valid: false }

Cómo se valida el RUC (módulo-11)

El último dígito del RUC es un check-digit calculado con módulo-11 sobre los primeros 10 dígitos.
  1. Multiplica cada dígito por su peso: [5, 4, 3, 2, 7, 6, 5, 4, 3, 2]
  2. Suma todos los productos
  3. expected = (11 - (sum % 11)) % 10
  4. Compara con el último dígito del RUC
function isValidRUC(ruc: string): boolean {
  if (!/^\d{11}$/.test(ruc)) return false;
  if (!['10', '15', '17', '20'].includes(ruc.slice(0, 2))) return false;

  const weights = [5, 4, 3, 2, 7, 6, 5, 4, 3, 2];
  const sum = weights.reduce((acc, w, i) => acc + w * Number(ruc[i]), 0);
  const expected = (11 - (sum % 11)) % 10;
  return Number(ruc[10]) === expected;
}

API completa

FunciónDescripción
isValidDNI(dni)True si el DNI tiene formato válido (8 dígitos, no trivial).
isValidRUC(ruc)True si el RUC pasa módulo-11 SUNAT.
tipoContribuyente(ruc)Devuelve 'natural', 'juridica', 'no-domiciliado', 'sucesion' o null.
validateDocumento(doc)Detecta si es DNI o RUC y devuelve resultado unificado.
